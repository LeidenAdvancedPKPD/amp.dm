# Changelog

## amp.dm 0.2.0

- Categorized the pkgdown reference section
- General optimization for testing
- Fixed issues for `attr_factor` to retain attributes and added option
  to set largest group first
  ([\#11](https://github.com/LeidenAdvancedPKPD/amp.dm/issues/11))
- Added function `flag_outliers` to flag observations for outliers based
  on box.plot stats
- Added function `check_cat` to check categories
- Added function `impute_covar` to impute covariates
- Added function `num_lump` to perform lumping of variables
- Added support in `general_tbl` to pass arguments to xtable
  ([\#3](https://github.com/LeidenAdvancedPKPD/amp.dm/issues/3))
- fixed bug for counting
  ([\#8](https://github.com/LeidenAdvancedPKPD/amp.dm/issues/8))
